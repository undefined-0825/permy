# Deploy Strategy（現時点：自動デプロイなし / MUST）
**Last Updated (JST):** 2026-03-02 04:35:00 +0900

## 1. 決定事項（MUST）
- 現時点では **自動デプロイを採用しない**（CIのみ）
- デプロイは手動で行う（仕様が固まれば頻繁な更新は想定しない）

## 2. 手動デプロイ手順（方針 / SHOULD）
- デプロイ先（例：Render/Cloud Run 等）は運用負荷が低いものを選ぶ
- 変更は小さく、デプロイ前にCIが通っていることを必須条件とする

## 3. 将来の拡張（SHOULD）
- 将来自動化する場合は、staging→prodを段階導入
- E2Eは手動トリガに限定し、OpenAI●●暴発を防ぐ
