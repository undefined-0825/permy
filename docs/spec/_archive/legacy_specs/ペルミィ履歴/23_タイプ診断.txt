# 診断ロジック定義書

## 1. タイプの定義

### 【夜の私】（営業スタイル）
| 英語表記 | 日本語表記 | 備考・特徴 |
| :--- | :--- | :--- |
| **MamePush** | **マメ営業タイプ** | 連絡頻度とレスポンスの速さで勝負。組数を稼ぎ、安定した指名を狙う。 |
| **Heal** | **癒しタイプ** | 聞き上手で居心地の良さを提供。お客様の精神的支柱となり、細く長く愛される。 |
| **LittleDevil** | **小悪魔タイプ** | 駆け引きや思わせぶりな態度で翻弄。お客様を依存させ、感情を動かして売上を作る。 |
| **BigClient** | **太客育成タイプ** | 特定の「エース」をプロデュース。一撃の単価を極大化させ、深い信頼と支援を得る。 |
| **Balance** | **バランスタイプ** | 空気を読む天才。場を盛り上げるノリと、深追いしない適度な距離感を両立。 |

### 【本当の私】（ライフスタイル・価値観）
| 英語表記 | 日本語表記 | 備考・特徴 |
| :--- | :--- | :--- |
| **Stability** | **安定重視タイプ** | 平穏な日常とメンタル維持を優先。波風を立てず、リスクを避けて働きたい。 |
| **Independence** | **自立タイプ** | 誰にも頼らず、自分の腕一本で稼ぐことに誇りを持つ。キャリア志向が強い。 |
| **Approval** | **承認欲求タイプ** | 人からの賞賛や「可愛い」「凄い」という言葉が原動力。華やかな世界が好き。 |
| **Realism** | **現実派タイプ** | 夢より数字と効率。私生活や別の仕事との両立を冷徹に計算するリアリスト。 |
| **Romance** | **ロマンタイプ** | 感情や直感を大切にする。人生にドラマを求め、運命的な繋がりを信じる。 |

## 2. 診断設問とポイント加算ロジック

回答に応じて各タイプにポイントを加算し、最終的に合計値が最も高いタイプに分類します。

### 【夜の私】設問案（全3問）

**Q1. 接客で一番「私、輝いてる！」と思う瞬間は？**
* **A：圧倒的なレスポンスと愛嬌で、お客様を喜ばせているとき**
  → [MamePush: +2] [Balance: +1]
* **B：深い話を聞いて、お客様がリラックスして満足そうにしているとき**
  → [Heal: +2] [BigClient: +1]
* **C：わがままを聞いてもらったり、お客様をドキドキさせているとき**
  → [LittleDevil: +2] [BigClient: +1]

**Q2. 理想的な売上の作り方はどっち？**
* **A：たくさんのお客様に呼ばれて、毎日コンスタントに指名がほしい**
  → [MamePush: +2] [Balance: +1]
* **B：特定の数名から大きな支援（シャンパン等）をガツンと受けたい**
  → [BigClient: +2] [LittleDevil: +1]
* **C：お酒も売上もそこそこに、ストレスなく細く長く続けたい**
  → [Heal: +2] [Balance: +1]

**Q3. お休みの日にお客様からLINEが。あなたの対応は？**
* **A：即レス！日常の何気ない会話で距離をグッと詰める**
  → [MamePush: +2] [BigClient: +1]
* **B：あえて少し時間を置いてから、思わせぶりな返信で焦らす**
  → [LittleDevil: +2] [Balance: +1]
* **C：基本は返さない。お店に来てくれた時に全力で癒やす**
  → [Heal: +2] [Realism(本当の私連動): +1]

---

### 【本当の私】設問案（全3問）

**Q1. 目標金額を達成！自分へのご褒美はどうする？**
* **A：将来のためにしっかり貯金。残った分で美味しいものを食べる**
  → [Stability: +2] [Realism: +1]
* **B：ずっと欲しかったハイブランドのバッグや靴を買いに行く！**
  → [Approval: +2] [Romance: +1]
* **C：誰にも頼らず自分の力で稼いだ証として、投資や自分磨きに使う**
  → [Independence: +2] [Realism: +1]

**Q2. ぶっちゃけ、今の仕事はいつまで続けるつもり？**
* **A：目標額が貯まるまで。効率よく稼いで次のステップ（昼職・起業等）へ**
  → [Realism: +2] [Independence: +1]
* **B：素敵な出会いやドラマチックな展開があれば、流れに身を任せたい**
  → [Romance: +2] [Approval: +1]
* **C：今の生活リズムを壊したくないし、必要とされる限りは安定して続けたい**
  → [Stability: +2] [Approval: +1]

**Q3. SNS（インスタやX）での自分、どう見られたい？**
* **A：「センスがいい」「可愛い」と、みんなに注目されてチヤホヤされたい**
  → [Approval: +2] [Romance: +1]
* **B：自立した一人の女性として、あるいは仕事のプロとして尊敬されたい**
  → [Independence: +2] [Realism: +1]
* **C：特に着飾らず、身近な友達やパートナーと平穏に繋がっていればいい**
  → [Stability: +2]

---

## 3. 分類アルゴリズム
1. ユーザーの回答に基づき、各カテゴリー内（夜職/本当）でポイントを合算。
2. **同点の場合の優先順位（例）:**
   - 夜職：Balance ＞ 他のタイプ（汎用性が高いため）
   - 本当：Stability ＞ 他のタイプ（ベースラインのため）
3. 最終的に「夜の私」×「本当の私」の組み合わせ（例：マメ営業×現実派）を決定。
