# 夜職向け返信アシストアプリ：最終仕様確定ドキュメント

## 1. 策定経緯と設計思想（ChatGPTへの共有事項）

本アプリは、単なるAIによる自動返信ではなく、夜職の現場における「営業戦略」と「個人の生活実態」を高度に両立させることを目的としている。議論の結果、以下の4つの設計思想を軸とした。

1. **二重構造のペルソナ（NightSelfType × RealSelfType）**
   - 営業スタイルとしての「夜の顔」と、生活背景や価値観としての「本当の顔」を掛け合わせる。これにより、「夜は小悪魔として振る舞うが、私生活ではパートナーとの時間を守り、深夜営業はしない」といった、リアルで持続可能な返信案の生成を可能にする。
2. **「思考プロセス（メタ指示）」と「NGルール」による制御**
   - 特定のキーワード（「会いたい」等）を無理に挿入する手法は、文脈を無視した不自然な「AI臭」を生むため棄却。代わりに、タイプごとの「視点（どう履歴を読むか）」と「NGルール（何を言ってはいけないか）」を定義することで、履歴に完全に同期しながらタイプ別のエッセンスを抽出する。
3. **呼称（呼び方）の動的判定とリスク回避**
   - LINE履歴から相手への呼び方を推測。ただし、履歴内の「第三者の名前（同僚や他のキャスト）」を自分への呼び方と誤認することを致命的なミスと定義。判定不能な場合は「あえて名前を呼ばない」というリスク回避を絶対ルールとした。
4. **「人間味」の徹底追及**
   - 丁寧すぎる敬語、過剰な句読点、高い漢字率を排除。改行を多用し、ひらがなを混ぜた「手打ち感のある短文」を追求する。

## 2. タイプの定義

### 【夜の私】（営業スタイル / NightSelfType）
| 英語表記 | 日本語表記 | 定義・役割 |
| :--- | :--- | :--- |
| **MamePush** | **マメ営業タイプ** | 圧倒的な連絡頻度とレスポンス。数多くの指名を獲得し、組数で売上を作る。 |
| **Heal** | **癒しタイプ** | 肯定、傾聴、包容力。お客様の精神的支柱となり、細く長く愛される。 |
| **LittleDevil** | **小悪魔タイプ** | 駆け引き、思わせぶり、甘え。手に入りそうで入らない距離感で客を翻弄する。 |
| **BigClient** | **太客育成タイプ** | 信頼と特別感。特定の「エース」をプロデュースし、一撃の高単価と深い支援を得る。 |
| **Balance** | **バランスタイプ（balance）** | 空気を読む天才。相手のテンションに120%同期し、居心地の良さを提供する。 |

### 【本当の私】（ライフスタイル・価値観 / RealSelfType）
| 英語表記 | 日本語表記 | 定義・役割 |
| :--- | :--- | :--- |
| **Stability** | **安定重視タイプ** | 平穏な日常とメンタル維持を優先。波風を立てず、リスクを徹底的に避ける。 |
| **Independence** | **自立タイプ** | 自分の力で稼ぐことに誇りを持つ。誰にも縛られないキャリア志向。 |
| **Approval** | **承認欲求タイプ** | 賞賛、注目、「可愛い」という評価が原動力。人から必要とされることで輝く。 |
| **Realism** | **現実派タイプ** | 効率、数字、結果。昼職や別の活動との両立を冷徹に計算するリアリスト。 |
| **Romance** | **ロマンタイプ** | 感情、直感、物語性。運命的な繋がりやドラマチックな展開を大切にする。 |

## 3. 最終診断設問およびポイント加算アルゴリズム

各回答に対し、該当するタイプへポイントを加算。最終的に合計値が最も高いタイプを採用する。

### 【夜の私】診断設問（NightSelfType）

**Q1. 接客で一番「私、輝いてる！」と思う瞬間は？**
- **A：圧倒的なレスポンスと愛嬌で、お客様を喜ばせているとき**
  → 加算：**MamePush(+2)**、**Balance(+1)**
- **B：深い話を聞いて、お客様がリラックスして満足そうにしているとき**
  → 加算：**Heal(+2)**、**BigClient(+1)**
- **C：わがままを聞いてもらったり、お客様をドキドキさせているとき**
  → 加算：**LittleDevil(+2)**、**BigClient(+1)**

**Q2. 理想的な売上の作り方はどっち？**
- **A：たくさんのお客様に呼ばれて、毎日コンスタントに指名がほしい**
  → 加算：**MamePush(+2)**、**Balance(+1)**
- **B：特定の数名から大きな支援（シャンパン等）をガツンと受けたい**
  → 加算：**BigClient(+2)**、**LittleDevil(+1)**
- **C：お酒も売上もそこそこに、ストレスなく細く長く続けたい**
  → 加算：**Heal(+2)**、**Balance(+1)**

**Q3. お休みの日にお客様からLINEが。あなたの対応は？**
- **A：即レス！日常の何気ない会話で距離をグッと詰める**
  → 加算：**MamePush(+2)**、**BigClient(+1)**
- **B：あえて少し時間を置いてから、思わせぶりな返信で焦らす**
  → 加算：**LittleDevil(+2)**、**Balance(+1)**
- **C：基本は返さない。お店に来てくれた時に全力で癒やす**
  → 加算：**Heal(+2)**

### 【本当の私】診断設問（RealSelfType）

**Q1. 目標金額を達成！自分へのご褒備はどうする？**
- **A：将来のためにしっかり貯金。残った分で美味しいものを食べる**
  → 加算：**Stability(+2)**、**Realism(+1)**
- **B：ずっと欲しかったハイブランドのバッグや靴を買いに行く！**
  → 加算：**Approval(+2)**、**Romance(+1)**
- **C：誰にも頼らず自分の力で稼いだ証として、投資や自分磨きに使う**
  → 加算：**Independence(+2)**、**Realism(+1)**

**Q2. ぶっちゃけ、今の仕事はいつまで続けるつもり？**
- **A：目標額が貯まるまで。効率よく稼いで次のステップへ行きたい**
  → 加算：**Realism(+2)**、**Independence(+1)**
- **B：素敵な出会いやドラマチックな展開があれば、流れに身を任せたい**
  → 加算：**Romance(+2)**、**Approval(+1)**
- **C：今の生活リズムを壊したくないし、必要とされる限りは安定して続けたい**
  → 加算：**Stability(+2)**、**Approval(+1)**

**Q3. SNS（インスタやX）での自分、どう見られたい？**
- **A：「センスがいい」「可愛い」と、みんなに注目されてチヤホヤされたい**
  → 加算：**Approval(+2)**、**Romance(+1)**
- **B：自立した一人の女性として、あるいは仕事のプロとして尊敬されたい**
  → 加算：**Independence(+2)**、**Realism(+1)**
- **C：特に着飾らず、身近な友達やパートナーと平穏に繋がっていればいい**
  → 加算：**Stability(+2)**

## 4. プロンプト生成・実装重要ルール

### 【全タイプ共通：重要事項】
1. **履歴最優先（ミラーリング）**: 直近のトーク履歴（熱量、語彙、改行リズム、絵文字量）を完全コピーせよ。
2. **呼称の厳密判定**: 履歴から相手への呼び方を推測。**第三者の名前との混同は厳禁。** 判定不能時は名前を呼ばず、不自然さを回避せよ。
3. **時間整合性**: 現在時刻と最終送信時刻の差を考慮し、自然な挨拶（おはよ、まだ起きてる？等）を生成せよ。
4. **人間味フィルタ**: 句読点（。、）を最小限化し、改行でリズムを作れ。漢字率を下げ、140文字以内の短文（1吹き出し）で完結させよ。
5. **新規客対策**: 履歴僅少時は、ユーザープロフ（方言・一人称）で補完しつつ、相手のプロフから話題を拾う「探り」の質問を生成せよ。

### 【タイプ別NGルール（裏設定）】
- **MamePush**: 返信催促、深刻な長文、過度な依存を誘発する重い言動の禁止。
- **Heal**: 相手への否定、説教、正論（ロジック）による解決提案の禁止。
- **LittleDevil**: ストレートすぎる愛の告白、過度な感謝、即レスすぎる雰囲気の禁止。
- **BigClient**: 安っぽい褒め、軽薄なノリ、他客（他者）との比較による優越感煽りの禁止。
- **Balance**: 自分の話の深掘り、一貫した強い主張、特定のキャラへの固定化の禁止。

### 【生成パターン】
以下の3つの意図に沿った返信案を提示すること。
1. **現状維持案**: 心地よいラリーを継続させる。
2. **来店誘導案**: 会いたくなる理由を添え、店への来店を促す。
3. **感情フック案**: 独占欲や承認欲求を刺激し、存在を意識させる。
